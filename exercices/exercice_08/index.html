Exercice 08

Cet exercice vous guide dans l'installation de Node.js et vous propose de créer votre première application avec OpenLayers


  1. Premièrement, télécharger Node.js ici
  2. Télécharger l'installateur, et laissez toutes les options par défaut
  3. Une fois Node.js installé, ouvrez Visual Studio Code et ouvrez le dossier de cet exercice (menu Fichier, Ouvrir un dossier)
  4. Ouvrez un terminal dans le dossier de cet exercice. Cela peut être fait depuis Visual Studio Code avec le menu "Terminal", puis "Nouveau terminal"
  5. Dans le terminal, tapez les commandes indiquées ici
    • npm create ol-app my-app indique à Node.js de créer une application de base OpenLayers. Node.js créera un nouveau dossier my-app contenant le code minimal pour faire fonctionner OpenLayers
    • cd my-app indique de déplacer le terminal dans le dossier my-app
    • npm start indique de démarrer l'application avec Vite. Vite va créer un serveur de développement executant l'application
    • Vite affichera dans la console une ligne semblable à Local: http://localhost:5173/, il s'agit de l'adresse à taper dans votre navigateur pour lancer l'application
  6. En vous rendant à l'adresse indiquée par Vite, vous devriez voir une carte OpenStreetMap en plein écran
  7. Le serveur de développement de Vite est très pratique. Vite actualisera automatique votre page web si vous modifier un des fichiers de l'application (fichier HTML ou JavaScript)
